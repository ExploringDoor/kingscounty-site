How Push Notifications Work

This page collects notification tokens and preferences. To actually send push notifications (e.g., from the admin page), a Firebase Cloud Functions backend is required. The notification infrastructure (service worker, token storage) is in place and ready for when the backend is set up.

Admin Setup Instructions

To activate push notifications:

  1. Go to Firebase Console > Project Settings > Cloud Messaging
  2. Under "Web Push certificates", click Generate key pair to get a VAPID key
  3. Replace YOUR_VAPID_KEY_HERE in this page's source code with the generated key
  4. Deploy a Cloud Function to send notifications using tokens from the notification_tokens Firestore collection
